# Kyanon-KFresh-Trainee-Test-2026

Repository này chứa bài làm của tôi cho bài kiểm tra tuyển dụng K-Fresh Trainee Program 2026 của Kyanon Digital.

## Cấu trúc dự án

Report.md  
Chứa báo cáo nghiên cứu so sánh giữa Magento và Drupal, công nghệ trong Tech Stack, và khái niệm Headless CMS.

Solution.php  
Chứa mã nguồn PHP để xử lý dữ liệu sản phẩm bao gồm:
- Lớp Product sử dụng lập trình hướng đối tượng (OOP)
- Lọc sản phẩm theo danh mục
- Áp dụng giảm giá cho các sản phẩm

README.md  
Cung cấp hướng dẫn cách chạy chương trình.

## Yêu cầu

- PHP 7 trở lên

## Cách chạy chương trình

1. Clone repository

git clone https://github.com/vokhacdoai280803-cloud/Kyanon-KFresh-Trainee-Test-2026.git

2. Di chuyển vào thư mục dự án

cd Kyanon-KFresh-Trainee-Test-2026

3. Chạy file PHP

php Solution.php

Kết quả sẽ hiển thị:
- Danh sách sản phẩm đã được lọc theo danh mục
- Danh sách sản phẩm sau khi áp dụng giảm giá
